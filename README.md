# GameAssetsSniffer
Categorize game assets and display them in the console, such as images, sound effects, and scripts.

# Usage

1. Open the webpage you want to sniff in your browser.

2. Press `F12` or right-click and select "Inspect" to open the browser's developer tools.

3. In the developer tools, navigate to the "Console" tab.

4. Copy the contents of the main.user.js file.
   
5. Press `Enter` to execute the script. Done!
